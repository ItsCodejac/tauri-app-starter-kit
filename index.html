Your desktop app's foundation. Replace this page with your frontend.

Make It Yours

1App name → tauri.conf.json productName
2Branding → src/lib/branding.js
3Menus → src-tauri/src/menu.rs custom_menus()
4Add your framework and replace this page

Add Your Frontend

# React
npm install react react-dom @vitejs/plugin-react

# Svelte
npm install svelte @sveltejs/vite-plugin-svelte

# Vue
npm install vue @vitejs/plugin-vue

# Or write vanilla JS directly

What's Working

Use the buttons above to open each utility window. Browse the documentation in the sidebar. Everything is functional — settings save, shortcuts persist, crash reports log, the tray works.

This page is index.html. Replace its contents with your app's UI. The sidebar, top bar, and status bar are part of the template — remove them too when you're ready. The Rust backbone and utility windows keep working regardless.
