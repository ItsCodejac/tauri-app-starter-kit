Make it yours

1Change app name → src-tauri/tauri.conf.json productName
2Edit menus → src-tauri/src/menu.rs custom_menus()
3Set branding → src/lib/branding.js
4Add your frontend framework and replace this page

Add your frontend

# React
npm install react react-dom @vitejs/plugin-react

# Svelte
npm install svelte @sveltejs/vite-plugin-svelte

# Vue
npm install vue @vitejs/plugin-vue

# Or just write vanilla HTML/JS in this file

Documentation

Full guides in docs/. Run mdbook serve docs or click Help → Documentation in the menu bar.

This is the template's default workspace. Replace the body of index.html with your app's UI. The Rust backbone and utility windows work regardless of framework.
