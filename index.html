Make it yours

1Change app name in src-tauri/tauri.conf.json → productName
2Edit menus in src-tauri/src/menu.rs → custom_menus()
3Set your branding in src/lib/branding.js
4Add your frontend framework and replace this page

Add your frontend

# React
npm install react react-dom @vitejs/plugin-react

# Svelte
npm install svelte @sveltejs/vite-plugin-svelte

# Vue
npm install vue @vitejs/plugin-vue

# Or just write vanilla HTML/JS in this file
